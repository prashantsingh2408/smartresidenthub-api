# **Smart Resident Hub API**  

## **API Endpoints**  

### **1. Sentiment Analysis**  

**Request Body:**  
- Lead chat data  
- RM comments  
- End-user feedback form  
- Call/audio recordings  

**Response:**  
- **Sentiment Score Categories:**  
  - Highly Interested  
  - Interested  
  - Moderate  
  - Not Interested  
  - Conversion Chance  
- **RM Rating**
